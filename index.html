    openligadb

    This project is an Scala implementation for the OpenligaDb API. At this point the documentation is not completed. When you can not find informaitons about the request you are interested in, take a look at the tests. At this Point there is an test for every implemented API request.

    What is OpenligaDB

    OpenligaDB is an communitybased Sports API.

    Quick start

    Add to build.sbt the following content:

    libraryDependencies += "com.quadstingray" % "openligadb" % "0.7.4"
